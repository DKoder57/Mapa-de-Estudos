# O que é Interface

[[UI-UX]] → Módulo 1

## Definição

Interface é o **ponto de conexão ou comunicação** que permite a troca de informações:

- Entre pessoas
- Entre pessoas e dispositivos
- Entre dispositivos

> [!NOTE] Origem O conceito se aplica a tecnologia, design, engenharia e interações humanas em geral.

---

## Tipos de Interface

### Interface Analógica

Botões físicos, painéis de controle mecânicos, relógios analógicos, rádios com sintonizador manual.

### Interface Digital

Sistemas operacionais, apps móveis, assistentes virtuais (Alexa, Siri), websites, smartphones/tablets, AR/VR.

### Interface Física (hardware)

Cabos, portas USB, conectores — ponto de contato entre componentes de hardware.

---

## Tipos de Interface de Usuário (UI)

|Sigla|Nome|Descrição|
|---|---|---|
|**GUI**|Graphical User Interface|Janelas, ícones, barras de rolagem — padrão em Windows/macOS|
|**CLI**|Command Line Interface|Interação por comandos textuais (terminal) — poderosa para usuários técnicos|

> [!TIP] GUI vs CLI A GUI substituiu em parte a CLI, mas ambas coexistem: CLI ainda é preferida por devs para automação e scripts.

---

## Objetivo de uma Interface

Uma interface bem projetada deve ser:

- **Clara** — o usuário entende o que fazer sem instruções externas
- **Acessível** — funciona para diferentes perfis de usuário
- **Intuitiva** — reduz a curva de aprendizado

Uma interface mal projetada gera confusão, erros e frustração, comprometendo o objetivo do sistema e a UX.
