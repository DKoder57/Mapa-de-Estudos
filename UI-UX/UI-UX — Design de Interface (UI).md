# Design de Interface (UI)

[[UI-UX]] → Módulo 1

## Definição

Design de interface é o processo de criar e organizar **elementos visuais e funcionais** que permitem a interação entre usuários e um sistema digital. Busca interfaces esteticamente agradáveis, fáceis de usar e que promovam uma experiência positiva.

---

## Componentes Visuais

|Elemento|Função|
|---|---|
|**Botões**|Elementos clicáveis que executam ações específicas|
|**Ícones**|Representações visuais para identificação rápida de funções|
|**Menus**|Organizam opções de navegação|
|**Tipografia**|Garante legibilidade e hierarquia via escolha de fontes e estilos|
|**Paleta de cores**|Define identidade visual e facilita navegação por contraste e destaques|

## Componentes Interativos

- **Formulários** — inserção de informações pelo usuário
- **Barras de navegação** — guias para exploração do sistema
- **Feedback visual** — respostas claras às ações (mudança de cor, animações, alertas)

---

## Princípios de Design de Interface

> [!NOTE] Os 9 princípios
> 
> 1. **Simplicidade** — evitar sobrecarga de informações
> 2. **Consistência** — elementos semelhantes têm mesmo comportamento em todo o sistema
> 3. **Visibilidade** — opções e ações possíveis são claramente identificáveis
> 4. **Hierarquia visual** — elementos organizados por importância (contraste, tamanho, cor, espaçamento)
> 5. **Acessibilidade** — uso por pessoas com diferentes habilidades e necessidades
> 6. **Feedback** — usuário informado sobre status, confirmações, erros
> 7. **Eficiência** — tarefas realizadas rápido e sem frustração, via atalhos e automações
> 8. **Responsividade** — adaptação a diferentes dispositivos e tamanhos de tela
> 9. **Estética** — design atraente aumenta engajamento

---

## Arquitetura Visual

Organiza e estrutura elementos para **guiar o olhar do usuário**, promovendo navegação intuitiva. Uma interface bem planejada:

- Direciona atenção para as áreas mais importantes
- Reduz sobrecarga cognitiva

---

## UI vs UX

> [!TIP] Distinção fundamental **UI** = elementos visuais e interativos (botões, cores, layouts) **UX** = experiência como um todo (o que o usuário sente, precisa e espera)
> 
> Uma interface bonita (UI) não garante boa experiência (UX), mas uma experiência fluida geralmente inclui uma interface bem projetada. O Design de Interface impacta diretamente em **engajamento, retenção e satisfação** com o produto.

## Guia de Estilos (prática no Projeto Desenvolve)

No projeto da Landing Page, o guia de estilos incluiu:

**Paleta de cores:** azul (confiança/profissionalismo), cinza (equilíbrio), preto (elegância/modernidade) — escolhidos por psicologia das cores, público-alvo, identidade de marca e testes de usabilidade.

**Tipografia:** fonte **Roboto** — versátil, legível em diferentes tamanhos de tela. Testada em 4 tamanhos (14px, 16px, 24px, 32px) e 2 pesos (Regular, Negrito) sobre azul, cinza, preto e branco. Critérios: legibilidade, hierarquia visual, estética e impacto emocional.

**Componentes:** campos de entrada, botões (com variações hover e inativo), ícones de redes sociais — catalogados como **Digital Design System**.
