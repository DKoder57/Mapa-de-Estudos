# ← [[Python]]

## Interpretador Python

Responsável por executar o código Python.

Exemplo:

```python
print("Olá Mundo")
```

Saída:

```text
Olá Mundo
```

---

## Python Interativo (REPL)

Executa comandos linha por linha.

```python
>>> 10 + 5
15

>>> 2 * 8
16
```

### Vantagens

- Testes rápidos
- Aprendizado inicial
- Experimentação

### Limitações

- Não salva histórico
- Não ideal para projetos

---

## Scripts Python

Arquivos com extensão:

```text
.py
```

Exemplo:

```python
print("Olá Mundo")
```

Execução:

```bash
python3 ola.py
```

## Visual Studio Code

IDE mais popular para Python.

Principais recursos:

- Destaque de sintaxe
- Terminal integrado
- Extensões
- Git integrado

> [!TIP]
> Instale as extensões Python e Jupyter logo após instalar o VS Code.

---

## Jupyter Notebook

Arquivos:

```text
.ipynb
```

Misturam:

- Texto
- Código
- Resultados
- Imagens

Exemplo:

```python
x = 10
x * 2
```

Resultado:

```text
20
```

## Google Colab

Versão online do Jupyter Notebook.

Características:

- Gratuito
- Executado na nuvem
- Não exige instalação
- Integração com Google Drive

> [!NOTE]
> O código é executado nos servidores da Google e não no seu computador.
